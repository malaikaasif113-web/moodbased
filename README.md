Mood Based Cooking Website
Project Description

Mood Based Cooking Website is a simple web project developed using HTML, CSS, and JavaScript. The purpose of this project is to suggest meal ideas based on the user's current mood. Users can click a mood button and receive food recommendations that match their selection.

This project demonstrates basic front-end development concepts including page structure, styling, and interactive functionality.

Features
Simple and clean user interface
Mood selection using buttons
Dynamic recipe suggestions
Responsive and easy-to-use layout
Uses separate HTML, CSS, and JavaScript files
Available Mood Categories
Happy

Chocolate Cake

How to Run the Project
Create a project folder.
Add the following files:
index.html
style.css
script.js
Copy the provided code into their respective files.
Save all files.
Open index.html in any web browser.
Working
Open the webpage.
Select a mood button.
The website processes the selected mood.
Food suggestions appear on the screen.
